# Personal Portfolio

A modern, responsive personal portfolio built with React.js. This site showcases my projects, technical skills, and research interests.

## Features
- **Modern UI**: Clean, responsive design with smooth animations.
- **Project Showcase**: A detailed list of my work, including the CGPA PathFinder and To-Do App.
- **Tech Stack**: Built using React, React Router (HashRouter for GitHub Pages), and custom CSS.
- **Easy Deployment**: Integrated with `gh-pages` for seamless updates.

## Local Setup

To run this project locally:

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```

## Projects Mentioned
- **CGPA PathFinder**: A tool for college students to plan their academic goals.
- **To-Do App**: A persistent task management utility.
- **Weather App**: Real-time weather data integration.
